# EVCC Token Renewal — Home Assistant Add-On Repository

Automatically renews [EVCC](https://evcc.io) trial sponsor tokens before they expire.

## Installation

1. In Home Assistant: **Settings → Add-ons → Add-on Store → ⋮ → Repositories**
2. Add: `https://github.com/cschnaible-1899/evcc-token-renewal`
3. Install **EVCC Token Renewal** from the store
4. Configure the add-on options (see add-on README)
5. Start the add-on
